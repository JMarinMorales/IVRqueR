# IV R que R

Repositorio para compartir material del IV congreso &amp; XV Jornadas de usuarios de R de Valencia (5-7 noviembre, 2025)

Este repositorio está destinado a compartir materiales relacionados con el IV Congreso & XV Jornadas de Usuarios de R de Valencia, que se celebró del 5 al 7 de noviembre de 2025. Aquí encontrarás presentaciones, documentos y otros recursos compartidos por los ponentes y participantes del evento.

## Estructura del Repositorio
- `Ponencias/`: Contiene las presentaciones utilizadas en las ponencias del congreso.
- `Documentos/`: Documentos adicionales relacionados con el congreso.
- `Fotos/`: Fotografías del evento.
